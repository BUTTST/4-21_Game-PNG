## 專案敘述
提供零依賴的【HTML5-Canvas】遊戲腳本，結合【JavaScript】開發流暢的本地化生存遊戲；並內建【SVG】到【PNG】的批次匯出能力，大幅簡化前端圖形處理與獨立遊戲的開發流程。

## 遊戲簡介
**《Arcane Survivor》** 是一款建立在現代瀏覽器，以 HTML5 / Canvas 與 JavaScript 打造的輕量化生存彈幕遊戲（吸血鬼倖存者 Like）。

* **故事背景**：玩家將化身為秘法大師，身處危機四伏的黑暗森林，驅逐四面八方湧動而來的魔物（史萊姆、暗影蝙蝠、石巨人、血色蜘蛛）。
* **核心玩法**：
  * **滑鼠/觸控自動施法**：跟隨你的游標移動，秘法大師將會自動並精準地朝反方向發射元素火球。
  * **魔力覺醒 (升級系統)**：擊退魔物後收集「魔力水晶」，累積滿後即可覺醒更強的魔法，如：`多重火球`、`烈焰貫穿`、`極速詠唱`、`風行步法`、`爆裂衝擊`。
  * **奧術護盾**：被擊中時消耗護盾值，若停止受傷一段時間將自動恢復魔力護盾。
