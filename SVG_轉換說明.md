# Neon Pulse 遊戲圖示與轉換工具

## 檔案說明
- `icon.svg`: 遊戲專用向量圖示 (SVG)，支援無限放大不失真。
- `icon_converter.html`: 零依賴的圖示轉換工具 (腳本)。

## 如何產生 PNG 圖檔
1. 在瀏覽器中直接點擊開啟（或拖曳） `icon_converter.html`。
2. 畫面會自動預覽 `icon.svg`（若無法自動載入，可點擊按鈕手動選擇檔案）。
3. 點擊 **"一鍵下載所有尺寸"** 按鈕。
4. 瀏覽器會自動下載遊戲開發所需的標準尺寸 PNG 圖片：
   - 16x16
   - 32x32
   - 64x64
   - 128x128
   - 256x256
   - 512x512

> **提示**：不需要安裝任何 Python 或 Node.js 模組，HTML Canvas 會自動處理轉換！
